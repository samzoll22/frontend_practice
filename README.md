# frontend_exercise_samzoll

# Start directly from browser

### unzip files and navigate to:

### `react-client > dist > index.html`

# Start with server

## To start open terminal and execute the following commands:

### `npm install`

### `npm run react-dev`

### `npm run server-dev`

## in your browser:

### `visit localhost:3000 || 127.0.0.1:3000`
